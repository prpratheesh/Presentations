01 // STACK PROFILE

Technical Foundation

Frontend Application

Built with Flutter for high-performance kiosk rendering. Interfaces directly with Adler APK950Pro hardware using Native Android Intents for 100% scan reliability without focus loss.

API Bridge

Powered by Node.js (API v2). A localized bridging server providing zero-config mDNS (Bonjour) discovery, REST endpoints, and bi-directional WebSocket broadcasting.

Enterprise Data Layer

Abstracted Data Providers supporting MSSQL Stored Procedures for direct DB access, or transparent integration with 3rd-party REST APIs like SAP/ERP.

02 // CAPABILITIES

Intelligent Features

  • Auto-Discovery: Zero-config setup. The app automatically resolves the API server on the local .local network.
  • Dynamic Media Scheduling: Full-screen image/video advertising with time-based interval controls (10s timeline cycles) pushed to idle screens.
  • Real-Time Configuration: Remote dashboard management of themes, layout templates, and timeout parameters synced instantly via WebSockets.
  • Emergency Broadcast: Capable of pushing instant alert messages to all active terminals simultaneously.

04 // INFRASTRUCTURE

Security & Networking

LAN Isolation

System operates purely on mDNS (Bonjour), restricting visibility entirely to the internal retail network layer.

SQL Protection

Database interactions rely exclusively on parameterized stored procedures, eliminating SQL injection vectors.

Encrypted Transport

Internal communication to SQL Server utilizes TLS/TDS encryption (DB_ENCRYPT flags).

Device Fingerprinting

Every kiosk generates a unique UUID to prevent unauthorized configuration overrides.
